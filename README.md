📝 To-Do List Web App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript.
This app allows users to add, complete, and delete tasks while storing data in the browser using Local Storage, ensuring tasks remain saved even after refreshing the page.

🚀 Features

➕ Add new tasks

✅ Mark tasks as completed

❌ Delete tasks

💾 Tasks stored using Local Storage

🔄 Tasks persist even after page refresh

🎨 Simple and responsive UI

🛠️ Technologies Used

HTML5 – Structure of the application

CSS3 – Styling and layout

JavaScript (ES6) – Functionality and DOM manipulation

Local Storage API – Data persistence in the browser

⚙️ How It Works

User enters a task in the input field.

Clicking Add Task adds the task to the list.

Tasks are saved in localStorage so they remain after refresh.

Clicking on a task toggles completed status.

Clicking Delete removes the task from both the UI and localStorage.
